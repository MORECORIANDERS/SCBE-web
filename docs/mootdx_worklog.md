## 代码示例

### 连接服务器

```python
from mootdx.quotes import Quotes

client = Quotes.factory(market='std', server=['110.41.147.114', 7709])
```

### 获取日线数据

```python
data = client.bars(symbol='113678', frequency=9, offset=1000)
```

### 获取行情详情

```python
detail = client.quotes('113678')
```

### 获取 F10 详细数据

```python
# 可转债 F10
f10_bond = client.F10('113678')
print(f10_bond.keys())  # 查看所有分类

# 正股 F10
f10_stock = client.F10('603220')
print(f10_stock.keys())  # 查看所有分类
```

## 注意事项

1. 部分服务器可能无法连接，需测试可用服务器
2. 扩展市场 (ext) 接口目前已失效
3. 日线数据需要通过 `frequency=9` 参数获取
4. DataFrame 转换为 JSON 时需处理 Timestamp 类型
