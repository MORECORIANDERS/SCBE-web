## 10. 完整示例代码

```python
from mootdx.quotes import Quotes
from mootdx.affair import Affair

# ============================================
# 1. 创建行情客户端
# ============================================
client = Quotes.factory(market='std')

# ============================================
# 2. 实时行情查询
# ============================================
# 春23转债
quotes_113667 = client.quotes(symbol='113667')
print('春23转债实时行情:', quotes_113667)

# 春秋电子
quotes_603890 = client.quotes(symbol='603890')
print('春秋电子实时行情:', quotes_603890)

# ============================================
# 3. K线数据获取
# ============================================
# 最近5条日K线
bars_113667 = client.bars(symbol='113667', frequency=9, offset=5)
bars_603890 = client.bars(symbol='603890', frequency=9, offset=5)
print('春23转债K线:', bars_113667)
print('春秋电子K线:', bars_603890)

# ============================================
# 4. 分钟数据
# ============================================
minute_603890 = client.minute(symbol='603890')
print('春秋电子分钟数据:', minute_603890.tail(10))

# ============================================
# 5. 指数数据
# ============================================
index_000001 = client.index(symbol='000001')
print('上证指数:', index_000001)

# ============================================
# 6. 财务数据
# ============================================
finance_603890 = client.finance(symbol='603890')
print('春秋电子财务数据:', finance_603890)

# ============================================
# 7. F10基本面数据
# ============================================
f10_603890 = client.F10(symbol='603890')
print('春秋电子F10:', f10_603890)

# ============================================
# 8. 服务器连接信息
# ============================================
print('最佳IP:', client.bestip)
print('服务器:', client.server)
print('超时:', client.timeout)

# ============================================
# 9. 财务数据文件
# ============================================
files = Affair.files()
print('财务文件列表:', files)

# 下载单个文件
Affair.fetch(filename='gpcw20260331.zip', downdir='tmp')

# 下载全部
Affair.parse(downdir='tmp')
```

## 附录: mootdx 主要功能速查表

| 功能类别 | 方法 | 说明 |
|---------|------|------|
| **实时行情** | `quotes(symbol)` | 获取实时行情数据 |
| **K线数据** | `bars(symbol, frequency, offset)` | 获取K线数据 |
| **分钟数据** | `minute(symbol)` | 获取分钟线数据 |
| **指数数据** | `index(symbol)` | 获取指数数据 |
| **财务数据** | `finance(symbol)` | 获取财务数据 |
| **基本面** | `F10(symbol)` | 获取F10数据 |
| **逐笔交易** | `transaction(symbol)` | 获取逐笔交易数据 |
| **股票数量** | `stock_count()` | 获取市场股票数量 |
| **所有股票** | `stock_all()` | 获取所有股票信息 |
| **服务器IP** | `bestip` | 获取最佳服务器IP |
| **财务文件** | `Affair.files()` | 获取财务文件列表 |
| **文件下载** | `Affair.fetch()` | 下载单个财务文件 |
| **批量下载** | `Affair.parse()` | 下载并解析全部财务数据 |
