# 可转债投资系统 - 产品需求文档

## 1. 产品概述

可转债投资系统是一款面向个人投资者的可转债行情分析工具，采用 GitHub 极简风格设计，为投资者提供可转债市场行情监控、双低筛选、行业分析等功能。

### 目标用户
- 个人可转债投资者
- 需要实时监控可转债市场的交易者

### 核心价值
- 实时展示可转债市场核心指标
- 提供双低（低价格、低溢价率）筛选工具
- 支持行业热力图和散点图分析
- 简洁高效的用户界面

## 2. 核心功能

### 2.1 页面架构

| 页面名称 | 功能描述 |
|---------|---------|
| 登录页面 | 用户认证入口，localStorage 模拟登录状态 |
| 首页 | 可转债市场核心指标展示，包含导航栏、指标卡片、功能入口、涨跌分布图、可转债列表 |
| 行业热力图 | 按行业维度展示可转债溢价率/双低数值分布 |
| 双低散点分析 | 散点图展示可转债价格与溢价率关系 |
| 转债详情页 | 单只可转债详细信息，包含历史价格走势 |
| 系统控制面板 | 数据采集管理界面 |
| 系统参数配置 | 飞书推送、定时任务等配置 |

### 2.2 首页模块详情

#### 2.2.1 顶部导航栏
- 使用 Ant Design Tabs 组件实现
- 包含：首页、行业热力图、双低分析、转债详情页入口
- 手机端自动折叠为底部导航栏

#### 2.2.2 核心指标卡片区
横向排列 3 张轻量卡片：
1. **全市场转债均价**
   - 展示数值
   - 涨跌幅（带颜色标识）
   - 迷你趋势线（ECharts 折线图）

2. **平均转股溢价率**
   - 展示数值
   - 涨跌幅
   - 迷你趋势线

3. **平均双低数值**
   - 展示数值
   - 涨跌幅
   - 迷你趋势线

#### 2.2.3 快捷功能入口区
横向排列 4 个功能入口：
1. 自选转债
2. 高级筛选
3. 强赎预警
4. 系统设置

#### 2.2.4 涨跌分布模块
- ECharts 柱状图展示涨跌幅分布
- 顶部统计：总成交额、涨跌家数、市场热度
- 柱状图区间：跌停、>7%、5-7%、3-5%、0-3%、平盘、0~-3%、-3~-5%、-5~-7%、<-7%

#### 2.2.5 可转债日报列表区
- Ant Design Table 组件
- 列：转债名称/代码、价格/涨跌幅、正股价格/涨跌幅、转股溢价率、剩余规模、双低数值
- 支持表头排序
- 手机端支持横向滚动

## 4. 响应式设计

### 4.1 断点设置

| 设备 | 断点 |
|------|------|
| 手机端 | < 768px |
| PC 端 | ≥ 768px |

### 4.2 手机端适配规则

- 导航栏自动折叠为底部固定导航栏
- 卡片纵向堆叠排列
- 功能入口改为横向滚动列表
- 表格支持横向滚动或转为卡片列表
- 图表保持全宽显示

## 5. 技术实现

### 5.1 技术栈

- **框架**: Vue3 + Vite
- **UI 组件库**: Ant Design Vue 4.x
- **路由**: Vue Router
- **图表**: ECharts
- **状态管理**: localStorage（模拟）
- **样式**: CSS + CSS Variables

### 5.2 路由定义

| 路由 | 页面 | 是否需要登录 |
|------|------|-------------|
| /login | 登录页面 | 否 |
| / | 首页（核心页面） | 是 |
| /heatmap | 行业热力图 | 是 |
| /scatter | 双低散点分析 | 是 |
| /detail/:code | 转债详情页 | 是 |
| /control | 系统控制面板 | 是 |
| /settings | 系统参数配置 | 是 |

### 5.3 数据模拟

所有页面数据使用 localStorage + 静态 JSON 模拟：
- 登录状态存储于 localStorage
- 配置参数存储于 localStorage
- 可转债行情数据使用静态 JSON 文件
